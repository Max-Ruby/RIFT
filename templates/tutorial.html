Most Explainable:

The most explainable methods allow you to provide an explanation for why you think an image is fake!

If you want an explanation you can give to a layman, use those first!

ELA

This highlights differences in JPEG compression rate by taking a difference of the original image and a compressed version.

In a real image, similar edges should have similar magnitudes. Similar textures - including flat surfaces - should have similar coloring.

In an image which has something spliced in, the spliced portion will often be much brighter or dimmer than the rest of the image!

Noise Analysis

This highlights differences in noise levels by taking a difference of the original image and a filtered image.

It's the same idea as the ELA - similar things in the image should have similar noise levels.

The different filters correspond to different kinds of noise! Some cameras are noisy in different ways.

The gaussian filter is great if you think the camera's kind of normal. This'll pick out the gaussian noise.

The median filter is great if you think the camera's super noisy. This'll pick out salt-and-pepper noise.

The bilateral filter is good if you think the image was compressed before being altered. This'll pick out compression artifacts.

Luminance Gradient

We plan on implementing the Luminance Gradient. This is a great bread-and-butter tool.

First, we can use it to detect if an image is AI generated - it amplifies high frequency noise, which is often present.

Second, similarly placed and angled objects should have similar luminance gradients.

Luminance Laplacian

This amplifies high frequency noise more. Can be useful sometimes, but is very situational.

Metadata Extraction

This picks up JPEG metadata, if it was available. It's a very situational tool - most people know to strip this out nowadays. When it hits, you can get real good information out.

Less Explainable:

These are more complex. They'll give you a clue, but not an explanation!

If these give a positive, then they're probably right! However, explaining why is difficult.
