General Note:

Please note that any output of this program is not definitive proof of anything.

These tools can be strong indicators that something is fake, but an absence of evidence is not evidence - it may fail to pick up a fake image.

Moreover, these tools can only determine if aaren image has been manipulated - these tools will never be able to determine if an image was staged or not.

That is, these tools will never be able to tell the difference between a very good stuffed dog and a real dog!
